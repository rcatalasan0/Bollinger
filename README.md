# Bollinger
Stock analysis using Bollinger band

### Steps (Windows)
1. Ensure you have Python installed (3.14.0 to be safe)
2. Install the required libraries (streamlit, plotly, yfinance, pandas). This can be done by being in the current directory and running the command "pip install -r requirements.txt"
4. Enter the directory of your desired version and run the command "python -m streamlit run app.py".

### (Linux)
1. Ensure you have Python installed
2. Enter the directory of your desired version and enter the command "source venv/bin/activate"
3. Acquire the necessary packages via "pip install pandas yfinance plotly streamlit"
4. Run program via the command "streamlit run app.py"

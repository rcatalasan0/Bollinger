# Bollinger
Stock analysis using Bollinger band

### Steps
1. Ensure you have Python installed (3.14.0 to be safe)
2. Install the required libraries (streamlit, plotly, yfinance, pandas)
  - This can be done by being in the current directory and running the command "pip install -r requirements.txt"
3. Enter the directory of your desired version and run the command "python -m streamlit run app.py"
  - This will open the web browser.
